## 5. 各科目の実施スケジュール

### 5.1 科目A-2

#### ルール⓪:問題の通し番号付け

管理用に問題に通し番号をつける。

- 2010年分の問1を1番とする
- 2011年分の問1は26番(毎年25問 × 1年 + 1)
- 毎年「25の倍数 + 1」の数字から通し番号を振る
- カード番号 c から原問への逆引き(月次計画の出力時に使用):
  - 年度 = 2010 + ⌊(c−1)/25⌋
  - 問番号 = ((c−1) mod 25) + 1
  - 例:c=27 → 問2(2011年)、c=51 → 問1(2012年)

#### ルール①:新規カードの導入

- 月〜土に1枚ずつ新しい番号を導入(週6枚ペース)
- 日曜は新規導入なし、復習のみ
- カード番号 c の導入日:

  ```
  導入日(c) = 7·⌊(c−1)/6⌋ + ((c−1) mod 6) + 1
  ```

- 例:
  - c=1 → 1日目(月)
  - c=7 → 8日目(月、日曜スキップ)
  - c=44 → 51日目(第8週・火)→ ちょうど52行目

#### ルール②:復習間隔は公比2

各番号の復習間隔は 1, 2, 4, 8, 16, 32 …日。
カード c は導入日 D を 0 回目として、

```
D, D+1, D+3, D+7, D+15, D+31, …日目(= D + 2^k − 1)
```

に出現する。

検算(カード「1」、導入1日目):

| 回     | 出現日 | 行 | 前回からの間隔 |
|--------|------:|---:|---------------:|
| 導入   | 1     | 2  | —              |
| 1回目  | 2     | 3  | +1             |
| 2回目  | 4     | 5  | +2             |
| 3回目  | 8     | 9  | +4             |
| 4回目  | 16    | 17 | +8             |
| 5回目  | 32    | 33 | +16            |

ぴったり一致する。他の番号でも同様。

- 例:カード3 → 行 4, 5, 7, 11, 19, 35
- 例:カード15 → 行 18, 19, 21, 25, 33

#### ルール③:行の書式

- その日に学ぶ番号を昇順(導入の古い順)に左から並べる
- 行末に置かれた番号がその日の新規カード。合図として2回繰り返す
- 日曜は新規がないので末尾の重複なし、復習だけが並ぶ

## 6. 週次レビュー

毎週日曜の終わりに30分で実施する。「測る → ふりかえる → うごかす」の3ステップ。

### 6.1 測る:定量指標で現状把握

| 指標 | 目標 | 計算方法 |
|------|------|----------|
| 科目A-2 実施率 | 100% | 実施カード数 / 計画カード数 |
| 科目A-2 初見正答率 | 70%以上 | 新規カードの正答数 / 新規カード数 |
| 科目A-2 復習正答率 | 90%以上 | 復習カードの正答数 / 復習カード数 |
| 科目B-1 設問正答率 | 70%以上 | 正答設問数 / 解答設問数 |
| 科目B-1 読解+解答時間 | 45分/問 以内 | 平均所要時間 |
| 科目B-2 完答率 | 100% | 字数規定+論点を満たした設問の割合 |
| 学習時間 | (各自設定) | 総学習時間 |

ポイントは「初見正答率」と「復習正答率」を**分けて測る**こと。前者は知識量、後者は定着度を表し、対処法が異なる。

### 6.2 ふりかえる:ミスをパターンで分類する

その週のミスを以下の3カテゴリに振り分け、件数を記録する。

1. **知識不足型**:そもそも知らなかった
2. **理解不足型**:知っていたが、設問の文脈と結びつかなかった
3. **ケアレス型**:読み落とし、字数オーバー、誤字、転記ミス

3週連続で同じカテゴリが上位なら、対策(教材の追加 / 解答プロセスの見直し / セルフチェック手順の導入)を打つ。

### 6.3 うごかす:翌週の打ち手を1〜3個に絞る

観察を**具体的な行動**に変換する。

- ❌「割り込みの理解を深める」(具体性なし)
- ⭕「火・木の科目B-1で『割り込み処理』を含む問題を優先選択する」

打ち手は最大3つ。これ以上は実行されない。

### 6.4 バッファ運用ルール

日曜のバッファ枠は以下の優先順位で使う。

1. 平日に未消化となった項目の実施
2. 復習正答率が80%未満だったカードの追加復習
3. その週で「分からなかった概念」のキャッチアップ

### 6.5 エスカレーション基準

以下のいずれかに該当した時点で、計画そのものを見直す。

- 2週連続で実施率80%未満
- 4週連続で同じ論点を間違えている

## 7. 月次レビュー

毎月末日(または最終日曜)に60〜90分で実施する。週次が「戦術」なら月次は「戦略」。

### 7.1 トレンドを見る:週次指標の月次集計

| 指標 | 第1週 | 第2週 | 第3週 | 第4週 | 傾向 |
|------|------|------|------|------|------|
| 科目A-2 初見正答率 | | | | | ↑/→/↓ |
| 科目A-2 復習正答率 | | | | | ↑/→/↓ |
| 科目B-1 設問正答率 | | | | | ↑/→/↓ |
| 科目B-2 完答率 | | | | | ↑/→/↓ |
| 学習時間 | | | | | ↑/→/↓ |

「↓」が続く指標は原因(教材難度の上昇 / 疲労 / 計画過密 / 単に運)を特定する。週次で見えなかった**長期トレンド**を月次で初めて検出する。

### 7.2 区分ごとの到達度判定

| 区分 | A | B | C |
|------|---|---|---|
| 科目A-2 | 初見正答率 80%以上 | 70〜80% | 70%未満 |
| 科目B-1 | 設問正答率 75%以上 | 60〜75% | 60%未満 |
| 科目B-2 | 全設問完答 + 字数規定達成 | 完答のみ | 未完答あり |

判定がCの区分には、「Bに上げるために必要な条件」を1行で書く。例:「科目B-1:火・木のB-1で『割り込み処理』を含む問題を優先選択する」。

### 7.3 弱点マップの更新

月初に持っていた弱点リストを以下に振り分ける。

- **解消**:同種の問題を3問連続で正答、または論点を自分の言葉で説明できる
- **改善中**:正答率が上向き
- **未着手**:対策を打てていない

未着手が3つ以上溜まったら、翌月の重点に少なくとも1つ含める。

### 7.4 残工数の逆算チェック

```
試験日までの残り週数 × 週あたり工数 = 残利用可能工数
```

これを未消化の計画工数と照合する。オーバーする場合の削減優先順位:

1. 「いっぺんに解く用」の過去2年分は**削らない**(本番形式の確認に必須)
2. 科目B-1の問題数を絞る
3. 科目A-2は出題頻度が低い論点を統合・スキップ

- 字数配分の感覚(時間内に書き切れるか)

### 7.5 翌月の重点を「やる/やらない」で宣言する

```
来月の重点(3つまで):
1. ○○(理由:△△)
2. ○○(理由:△△)
3. ○○(理由:△△)

来月やらないこと:
- ○○(理由:△△)
```

「やらないこと」を明示するのが要点。新しいことを足し続けると計画が崩れる。

## 10. 月次計画の出力フォーマット

月次計画(月初に作成、月内の各日のやることを記す)は以下のルールで出力する。

### 10.1 構成

- 月間サマリ(期間・稼働日数・月間達成目標)
- 日別スケジュール(リスト形式、改行区切り)
- これ以外のセクション(週次内訳・レビュー留意点・運用ルールの再掲など)は含めない

### 10.2 日別エントリの形式

各日は以下の形式で記述する。

```
M/D (曜) 区分
- 科目A-2:問X(西暦)[導入/復習をその日の出題順に列挙、末尾が新規]
- 科目B-1:年度(期) 問X (実施日のみ。題材は実施後にメモへ)
- 科目B-2:年度(期) 問X「テーマ」工数
```

例:

```
5/11 (月) 稼働
- 科目A-2:問1(2010)導入
- 科目B-2:平成21年度秋期 問3「組込みシステムにおける適切な外部調達について」状況組み立て1

5/12 (火) 稼働
- 科目A-2:問1(2010)復習, 問2(2010)導入
- 科目B-1:平成22年度春期 問1
- 科目B-2:平成21年度秋期 問3「組込みシステムにおける適切な外部調達について」状況組み立て2

5/17 (日) バッファ
- 科目A-2:問4(2010)復習, 問6(2010)復習
```

### 10.3 表示の原則

- 通し番号 c は内部計算用。出力時は§5.1ルール⓪の逆引き式で「問X(西暦)」形式に変換する
- 科目B-2は年度・問番号に加え**テーマを必ず併記**し、何の問題かを一見で識別できるようにする
- 科目B-1はIPA公式題目がないため年度・問番号のみで識別。題材は実施後に学習記録(§8)へメモする
- 日曜(バッファ)も復習があれば列挙する。何もない日は項目を省略してよい
